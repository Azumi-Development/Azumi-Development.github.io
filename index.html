What is Azumi?


Azumi is a Discord bot shaped and made by the community. It is written in Python v3, with the help of Pycord v2.
Azumi is focused on being a widely available and easy-to-use and customize, utility bot. It has several features to help with your day-to-day Discord use, such as:


• Timestamp Generation
• Moderation Commands
• Weather Information
• Member Data
• GitHub Information
• Message Grabbing
• Math Calculation
• Dice Rolling
• Pronoun Role Managament (Beta)
• Server Logs (Coming Soon!)
